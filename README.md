## What is Katana CG Scene Lighting?

![Katana Lighting Interface](https://www.sweetwater.com/sweetcare/media/2024/06/Mod-view-of-Boss-Tone-Studio-for-Katana-Gen-3.jpg)

Katana CG Scene Lighting is a professional application built for lighting artists in the computer graphics industry. It offers a robust set of tools for designing, manipulating, and rendering complex light setups in 3D scenes. With support for PBR workflows, real-time feedback, and deep compositing integration, Katana streamlines the lighting process for high-end production pipelines.

The software is known for its scalability and flexibility, handling everything from simple product shots to full-length animated features. Its procedural approach allows artists to make changes late in the production process without sacrificing render time or quality. Katana also includes advanced features such as light path expressions, AOV management, and lookdev baking.

Ideal for use in film, television, game development, and architectural visualization, Katana CG Scene Lighting empowers artists to achieve cinematic results with technical precision and creative freedom. It supports industry-standard formats and renderers including Arnold, RenderMan, and V-Ray.

## Tips & Tricks

- Use light linking to control which objects are affected by specific lights  
- Save and reuse lighting setups across projects  
- Utilize render layers for multi-pass compositing  
- Enable real-time preview to see changes instantly
